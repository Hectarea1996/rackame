procedure

(rkm-create-command-buffer device    
  family-symbol) → rkm-command-buffer?
  device : rkm-device?
  family-symbol : symbol?
Crea un command buffer para la familia de colas representada por family-symbol.

El valor family-symbol puede ser:

  • 'graphics : Colas de tipo graphic.

  • 'transfer : Colas de tipo transfer.

  • 'compute : Colas de tipo compute.

  • 'present : Colas de tipo present.
