1 Instancia

procedure

(create-instance validation) → instance?

  validation : boolean?
Crea una instancia de vulkan.

Si validation no es #f se activará la capa de validación de vulkan.

procedure

(instance? v) → boolean?

  v : any
Comprueba si el valor v es una instancia. En ese caso se devuelve #t. En otro caso se retorna #f.

procedure

(get-instance-layers instance) → (listof bytes?)

  instance : instance?
Devuelve las capas activadas por la instancia.

procedure

(get-instance-extensions instance) → (listof bytes?)

  instance : instance?
Devuelve las extensiones activadas por la instancia.

procedure

(destroy-instance instance) → void?

  instance : instance?
Destruye una instancia de vulkan.
