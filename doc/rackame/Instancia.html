2 Instancia

struct

(struct rkm-instance (vk-instance layers extensions)
    #:extra-constructor-name make-rkm-instance)
  vk-instance : VkInstance
  layers : (listof bytes?)
  extensions : (listof bytes?)
Estructura que representa una instancia de vulkan.

layers es la lista de layers activadas. extensions es la lista de extensiones activadas.

procedure

(rkm-create-instance validation) → rkm-instance?

  validation : boolean?
Crea una instancia de vulkan.

Si validation no es #f se activará la capa de validación de vulkan.
